# Homework #4 - LLM API Experiment

## Overview
This project demonstrates how to interact with a Large Language Model (LLM) using code instead of a web interface. The OpenAI API is used to send prompts and analyze responses.

## Objectives
- Connect to an LLM API using Python
- Send multiple prompts
- Inspect and compare outputs
- Understand how LLMs behave with different inputs

## Files
- `main.py`: Python script that sends prompts to the LLM
- `requirements.txt`: Dependencies required to run the project
- `sample_outputs.txt`: Generated outputs from the model

## Prompts Used
- Explanation (reinforcement learning)
- Creative (haiku)
- Analytical (pros/cons of APIs)
- Beginner-friendly explanation (overfitting)
